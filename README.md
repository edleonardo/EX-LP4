## Install

```sh
yarn
```

## Usage

```sh
yarn dev
```

## Routes

* /produtorio/recursivo
```sh
body: {
  m(number),
  n(number)
}
```

* /produtorio/iterativo
```sh
body: {
  m(number),
  n(number)
}
```

* /fatorial
```sh
body: {
  num1(number),
  num2(number)
}
```
